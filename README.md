# Cuidados Pessoais | Bem-Estar e Saúde Domiciliar

## Sobre o Projeto

Este repositório apresenta a *Landing Page* desenvolvida para o serviço **Cuidados Pessoais**, especializado em assistência domiciliar, acompanhamento de pacientes e procedimentos técnicos de enfermagem na região de Campo Mourão — Paraná.

O projeto foi concebido com foco na construção de uma presença digital profissional, priorizando elementos essenciais para o segmento da saúde, como:

* credibilidade;
* humanização;
* acessibilidade;
* clareza de informações;
* facilidade de contato.

A plataforma foi estruturada estrategicamente para proporcionar uma experiência de navegação intuitiva e transmitir segurança ao usuário, facilitando o processo de agendamento e contato direto com a equipe responsável.

---

# Objetivos da Plataforma

A aplicação possui como principais objetivos:

* fortalecer a presença digital do serviço;
* transmitir profissionalismo e confiança;
* facilitar o agendamento de atendimentos;
* melhorar a comunicação com potenciais pacientes e familiares;
* otimizar o fluxo de conversão através da integração com WhatsApp.

---

# Tecnologias Utilizadas

O projeto foi desenvolvido utilizando tecnologias Front-End nativas, priorizando desempenho, organização estrutural e carregamento otimizado.

## HTML5

Responsável pela estrutura semântica da aplicação, garantindo melhor acessibilidade, organização de conteúdo e compatibilidade com mecanismos de busca.

---

## CSS3

Utilizado para construção visual da interface, incluindo:

* CSS Grid Layout;
* Flexbox;
* variáveis globais (`:root`);
* Media Queries;
* animações suaves e efeitos de interação.

A identidade visual foi planejada utilizando uma paleta de cores corporativa baseada em tons de azul e ciano, buscando transmitir sensação de tranquilidade, cuidado, limpeza e segurança.

---

## JavaScript (Vanilla)

Toda a interatividade da aplicação foi implementada em JavaScript puro, sem dependência de frameworks externos.

Entre os recursos desenvolvidos, destacam-se:

* menu responsivo para dispositivos móveis;
* cabeçalho dinâmico durante a rolagem;
* navegação com rolagem suave;
* integração automatizada com WhatsApp;
* manipulação dinâmica de elementos da interface.

---

# Principais Funcionalidades

## Integração Inteligente com WhatsApp

O formulário da plataforma captura informações do usuário, como:

* nome;
* telefone;
* serviço desejado.

Após o preenchimento, o sistema formata automaticamente uma mensagem personalizada direcionada ao WhatsApp da empresa, agilizando o processo de atendimento e contato inicial.

---

## Design Responsivo

A interface adapta-se automaticamente para diferentes dispositivos, incluindo:

* smartphones;
* tablets;
* notebooks;
* computadores desktop.

O objetivo é garantir acessibilidade e boa experiência de navegação em qualquer resolução de tela.

---

## Menu Mobile Otimizado

Foi desenvolvido um sistema de navegação lateral responsiva para dispositivos móveis, acompanhado de efeito de desfoque (*overlay*), proporcionando maior foco visual e melhor experiência de utilização em telas menores.

---

## Interface Voltada para o Segmento da Saúde

Toda a estrutura visual da aplicação foi construída com foco em princípios de UI/UX aplicados ao setor de saúde.

Foram considerados fatores como:

* legibilidade;
* organização visual;
* conforto de navegação;
* hierarquia de informações;
* psicologia das cores aplicada à área médica e assistencial.
